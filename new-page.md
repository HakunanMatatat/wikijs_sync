● 变压器（Transformer）

  一、什么是变压器

  变压器是一种利用电磁感应原理，将交流电的电压升高或降低的静止电气设备。它本身不产生电能，只进行电压和电流的变换。

  二、工作原理

  变压器基于法拉第电磁感应定律：

  1. 交流电流过一次绕组（初级线圈），产生交变磁通
  2. 交变磁通通过铁芯耦合到二次绕组（次级线圈）
  3. 二次绕组中感应出电动势，实现电压变换

  基本公式

  U₁ / U₂ = N₁ / N₂ = I₂ / I₁

  - U₁、U₂：一次、二次电压
  - N₁、N₂：一次、二次绕组匝数
  - I₁、I₂：一次、二次电流

  匝数比大于 1 为降压变压器，小于 1 为升压变压器。

  三、基本结构
  ┌────────────────┬──────────────────────────────────────────────┐
  │      部件      │                     作用                     │
  ├────────────────┼──────────────────────────────────────────────┤
  │ 铁芯           │ 提供磁通回路，由硅钢片叠压而成，减少涡流损耗 │
  ├────────────────┼──────────────────────────────────────────────┤
  │ 绕组           │ 一次绕组接电源，二次绕组接负载               │
  ├────────────────┼──────────────────────────────────────────────┤
  │ 油箱（油浸式） │ 绝缘和散热                                   │
  ├────────────────┼──────────────────────────────────────────────┤
  │ 套管           │ 绕组引线穿出油箱的绝缘装置                   │
  ├────────────────┼──────────────────────────────────────────────┤
  │ 冷却装置       │ 风冷、油冷或水冷，保证运行温度               │
  └────────────────┴──────────────────────────────────────────────┘
  四、分类

  按用途

  - 电力变压器 — 电网输配电，升压/降压
  - 配电变压器 — 将高压降为用户用电电压（如 10kV → 380V/220V）
  - 仪用变压器 — 电压互感器（PT）、电流互感器（CT），用于测量和保护
  - 特种变压器 — 电炉变压器、整流变压器、矿用变压器等

  按冷却方式

  - 干式变压器 — 空气冷却，适用于室内
  - 油浸式变压器 — 变压器油冷却，适用于大容量、室外

  按相数

  - 单相变压器
  - 三相变压器

  五、主要参数
  ┌─────────────────┬───────────────────────────────────────┐
  │      参数       │                 说明                  │
  ├─────────────────┼───────────────────────────────────────┤
  │ 额定容量（S）   │ 单位 kVA 或 MVA，表示变压器的输出能力 │
  ├─────────────────┼───────────────────────────────────────┤
  │ 额定电压        │ 一次侧和二次侧的设计电压              │
  ├─────────────────┼───────────────────────────────────────┤
  │ 额定电流        │ 对应额定容量下的电流                  │
  ├─────────────────┼───────────────────────────────────────┤
  │ 空载损耗（P₀）  │ 铁芯中的磁滞和涡流损耗                │
  ├─────────────────┼───────────────────────────────────────┤
  │ 负载损耗（Pₖ）  │ 绕组电阻产生的铜损                    │
  ├─────────────────┼───────────────────────────────────────┤
  │ 短路阻抗（Uₖ%） │ 反映变压器内阻抗大小                  │
  ├─────────────────┼───────────────────────────────────────┤
  │ 效率（η）       │ 一般在 95%～99% 以上                  │
  └─────────────────┴───────────────────────────────────────┘
  六、损耗与效率

  η = P₂ / P₁ × 100%
    = P₂ / (P₂ + P₀ + β²Pₖ) × 100%

  - β = 负载系数（实际负载 / 额定容量）
  - 当 P₀ = β²Pₖ 时，效率最高

  七、常见应用场景

  - 发电厂 — 发电机出口升压（如 10.5kV → 220kV / 500kV）
  - 变电站 — 逐级降压输送到用户
  - 工厂/小区 — 配电变压器降至 380V/220V
  - 电子设备 — 小型变压器用于电源适配器
  - 电焊机 — 降压大电流输出

  八、发展趋势

  - 非晶合金铁芯 — 空载损耗降低 60%～80%
  - 智能变压器 — 集成传感器和通信模块，实现在线监测
  - 固态变压器（SST） — 基于电力电子技术，体积小、可控性强
  - 高温超导变压器 — 大幅减小体积和损耗
